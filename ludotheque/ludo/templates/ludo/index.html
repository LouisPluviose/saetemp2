Présentation du projet


Ce projet est un projet de développement d'un site web. Il est développé en utilisant le framework django. Nous l'avons réalisé en groupe pour la SAE23.


Tâches que nous avons réaliser:

  • Django (models, views, template et static
  • Migration de la base de données sqlite3 sur une base de donnée externe en sql


Tâches que nous n'avons pas réaliser :

  • Mise en ligne des images
  • Liste des jeux
  • Mise en place du wsgi et du serveur apache
